# todo-ext

Extensión de VSCode que busca y lista comentarios `TODO` pendientes del proyecto en un panel dedicado en la Activity Bar.

## Features

- Vista de árbol "Tareas Pendientes" en su propio contenedor de la Activity Bar (`TODO Finder`).
- Comando `Hello World` (`todo-ext.helloWorld`) de ejemplo.

## Desarrollo

```bash
pnpm install
pnpm run watch
```

Lanza con `F5` para abrir un Extension Development Host.
